## What is Web A.0?

Every era of the internet expanded who could participate:

- **Web 1.0** — Read (consume)
- **Web 2.0** — Read + Write (create)
- **Web 3.0** — Read + Write + Own (decentralize)
- **Web A.0** — Read + Write + **Agent** (act autonomously)

Web A.0 is the era where AI agents are first-class participants. ATLAST Protocol is the trust layer that makes it possible.
